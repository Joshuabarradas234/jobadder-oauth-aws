JobAdder OAuth on AWS · Security & Data Flow Overview
AWS eu-west-2 · London
Version 1.3 · April 2025 Confidential
Overview: JobAdder OAuth connects to JobAdder via a secure, GDPR-compliant integration hosted entirely within AWS eu-west-2 (London). Candidate data is retrieved, processed by the the app AI agent, and written back to JobAdder — all within a single, time-bounded operation. No candidate data is stored at rest outside of JobAdder. All sub-processors are disclosed below and operate under signed Data Processing Agreements.

Security & Identity
AWS WAF
Protects all inbound connections. Blocks malicious traffic before it reaches the platform.
AWS KMS
AES-256 encryption for all credentials and tokens at rest.
Secrets Manager
Secure credential storage with automatic rotation. Never exposed in code or logs.
Processing
API Gateway
Receives Partner Action callbacks from JobAdder. Validates and routes securely.
Serverless Compute
Fetches, processes, and writes back the CV. Ephemeral — no data persisted after execution.
Queue & Retry
Ensures reliable delivery. Retries on failure. Failed jobs escalate to engineering automatically.
